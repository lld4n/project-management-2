# Skill: timeline_insights

## Назначение

Используется для аккуратного выявления трендов и изменений по времени.

## Когда применять

- weekly trends
- period comparison
- stable vs short-lived patterns
- shifts in top artists / genres

## Правила

- опираться только на factual summaries;
- различать устойчивый паттерн и краткосрочный всплеск;
- не делать психологических выводов;
- не выдавать интерпретацию как факт.

## Допустимые формулировки

- "в данных видно"
- "по наблюдаемым значениям"
- "в этот период чаще встречалось"
- "можно осторожно предположить"

## Недопустимые формулировки

- "пользователю надоело"
- "настроение ухудшилось"
- "это доказывает"
- "очевидно"

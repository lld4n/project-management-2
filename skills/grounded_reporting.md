# Skill: grounded_reporting

## Назначение

Используется report agent-ом для финального ответа пользователю.

## Цель

Сделать ответ:

- кратким;
- factual;
- без выдумывания;
- с указанием ограничений, если нужно.

## Правила

- использовать только approved claims;
- не восстанавливать rejected claims;
- если данных мало, прямо об этом сказать;
- не добавлять личностные интерпретации;
- отделять факт от более слабой интерпретации.

## Формат

- 1 короткий абзац для простого factual ответа;
- 2 коротких абзаца для аналитического ответа;
- без лишней болтовни.

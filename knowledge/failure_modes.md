# Failure Modes

## Типовые ошибки системы

1. Модель отвечает без расчёта.
2. Insight agent делает слишком смелую интерпретацию.
3. Verifier пропускает claim, который звучит правдоподобно, но не доказан.
4. Report agent добавляет уверенность, которой нет в данных.
5. Rolling snapshots трактуются как независимые наблюдения.

## Что делать

- ужесточать verifier;
- использовать canonical layer;
- сокращать свободу report agent;
- предпочитать factual route, если аналитический route не нужен.
